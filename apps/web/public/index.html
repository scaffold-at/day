scaffold-daypre-release

Scaffold your day with AI.

A CLI-first, MCP-ready scheduler that places your TODOs into the free slots of your calendar — with AI clients (Claude Code, Cursor, Claude Desktop) as a first-class user. Same surface via CLI flags or MCP tools.

Install

curl -fsSL https://day.scaffold.at/install.sh | sh

v0.1 binaries are unsigned (Apple notarization lands post-v0.1). The installer clears the macOS quarantine attribute for you. Tier 1 targets: macOS Apple Silicon, Linux x64.

Quick start

scaffold-day init
scaffold-day today --tz Asia/Seoul
scaffold-day docs --for-ai      # AI-readable surface dump
