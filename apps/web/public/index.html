scaffold-dayv0.1 · pre-release

Scaffold your day with AI.

A CLI-first, MCP-ready scheduler that places your TODOs into the free slots of your calendar — with AI clients (Claude Code, Cursor, Claude Desktop) as a first-class user. The same surface is exposed both as CLI flags and as MCP tools, so an LLM can plan your day with one tool call instead of piping CLI text.

Install

curl -fsSL https://day.scaffold.at/install.sh | sh

Tier 1: macOS Apple Silicon, Linux x64. v0.1 binaries are unsigned (Apple notarization is post-v0.1). The installer clears the macOS quarantine attribute for you. Pin a version with sh -s -- --version v0.1.0.

Quick start

$ scaffold-day init                  # ~/.scaffold-day + balanced policy
$ scaffold-day today --tz Asia/Seoul
$ scaffold-day docs --for-ai          # AI-readable surface dump

What it looks like

scaffold-day today ───────────────────────────────────────────── 2026-04-27 (Mon) · Asia/Seoul · policy: balanced events 10:00–11:00 Standup 14:00–15:00 Design review placements 09:00–10:00 Draft Q2 OKR doc ★ 78 11:30–12:30 Review PR #142 ★ 64 free slots 13:00–14:00 (1h) · 15:00–18:00 (3h)

Use it from your AI client

scaffold-day mcp speaks the Model Context Protocol over stdio — 24 tools (get_day, suggest_placement, place_todo, compute_task_importance, replan_day, explain_placement, …). Drop one of the snippets below into your client config:

~/Library/Application Support/Claude/claude_desktop_config.json

{
  "mcpServers": {
    "scaffold-day": {
      "command": "scaffold-day",
      "args": ["mcp"]
    }
  }
}

One command (writes ~/.config/claude-code/mcp.json or merges)

claude mcp add scaffold-day -- scaffold-day mcp

Cursor → Settings → MCP → Add

{
  "mcpServers": {
    "scaffold-day": {
      "command": "scaffold-day",
      "args": ["mcp"]
    }
  }
}

Tip: paste AGENTS.md into the system prompt at session start. It carries the JTBD recipes ("a meeting moved — replan") so the AI doesn't have to derive call sequences from scratch.

Why

Most calendar tools optimise for displaying time. scaffold-day optimises for placing time: take a list of TODOs, score their importance, and slot them into the calendar's actual free space — under a transparent, replayable policy you can edit by hand.
