# Code of Conduct

Scaffold Day adopts the [Contributor Covenant, version 2.1][cc-v21] as
its code of conduct. The full text is the canonical source and is
incorporated here by reference.

[cc-v21]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## In short

We pledge to make participation in this project a harassment-free
experience for everyone, regardless of background or identity. We
expect all participants to act with respect and good faith, and to help
maintain a welcoming environment for newcomers.

The full Contributor Covenant defines the specific standards of
acceptable and unacceptable behavior, the enforcement responsibilities
of maintainers, and the consequences of violations. Please read it
before contributing.

## Scope

This Code of Conduct applies within all project spaces, including:

- The GitHub repository (issues, pull requests, discussions, code review)
- Project chat channels and community forums
- When an individual is officially representing the project in public

It also applies to behavior outside these spaces when it has a direct
impact on the safety or well-being of project participants.

## Reporting

If you experience or witness behavior that violates this Code of
Conduct, please report it privately to:

- **Email**: `conduct@scaffold.at`
- **Subject prefix**: `[scaffold-day coc]`

All reports will be reviewed and investigated promptly and fairly.
The maintainer is obligated to respect the privacy and safety of the
reporter.

If a report concerns the maintainer themselves, please contact
`conduct-escalation@scaffold.at` (a separate inbox monitored by an
external advisor) so the conflict of interest can be handled
appropriately.

## Enforcement

Maintainers will follow the Contributor Covenant 2.1
[Enforcement Guidelines][cc-enforce] when determining the consequences
of any action they deem in violation:

1. Correction
2. Warning
3. Temporary ban
4. Permanent ban

[cc-enforce]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines
